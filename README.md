# Website Tra Cứu Tương Tác Thuốc - Bệnh Cho Phòng Khám

Hệ thống hỗ trợ bác sĩ tra cứu tương tác giữa thuốc và bệnh nhằm giảm thiểu rủi ro trong quá trình kê đơn.

- Đăng nhập

- Quản lý thuốc
- Quản lý bệnh
- Kiểm tra tương tác thuốc - bệnh
- Lịch sử tra cứu
